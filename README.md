# 🔬 MediScan AI-Driven Diagnostics Lab Analysis

A data-driven evaluation and interactive simulation of implementing an AI-powered diagnostic system for **MediScan Diagnostics**. This project analyzes the ROI, operational efficiency, and risk mitigation of transitioning from manual pathology reporting to an AI-integrated workflow.
LINK:https://business-studies-project-jjzk27dmcakoxruttdxvvb.streamlit.app/
---

## 📌 Project Overview

MediScan Diagnostics faces challenges with report delivery delays, human error in analysis, and rising operational costs. This project provides:
1.  **Financial Modeling:** Calculation of annual savings, Net Benefit, and ROI.
2.  **Operational Simulation:** Synthetic data generation to model processing time improvements.
3.  **Interactive UI:** A Streamlit-based application to perform "What-If" sensitivity analysis.

## 📊 Key Business Metrics (Baseline)
Based on an initial investment of **₹3 Crores**:
* **Time Reduction:** 50% faster processing.
* **Error Reduction:** 40% decrease in diagnostic inconsistencies.
* **Annual Savings:** ₹3.5 Crores.
* **Net Benefit (Year 1):** ₹50 Lakhs.
* **ROI:** 116.67%.

---

## 🚀 The Streamlit Dashboard

The project includes an interactive UI (`app.py`) that allows stakeholders to adjust variables in real-time.

### Features:
* **Dynamic Parameter Input:** Adjust investment, volume, and AI efficiency via a sidebar.
* **Live Metrics:** Instant updates on ROI and Breakeven timelines.
* **Comparative Visuals:** * *Time Distribution:* Shows the shift in diagnostic speed.
    * *Cost Structure:* Compares labor and risk costs between Manual and AI processes.
    * *Cumulative Benefit:* Projects profitability over a 3-year horizon.

### How to Run:
1. Install requirements: `pip install streamlit pandas numpy matplotlib seaborn`
2. Launch the app: `streamlit run app.py`

---

## 📈 Visual Analysis Explained

### 1. Processing Time Comparison
By applying a Gaussian distribution to manual processing times, the model illustrates how AI shifts the median turnaround time from 120 minutes to 60 minutes, effectively doubling lab capacity.

### 2. Sensitivity Analysis
The bar charts demonstrate project resilience. It identifies the "critical efficiency threshold"—the minimum AI performance required to maintain a positive Net Benefit in Year 1.

### 3. Risk vs. Cost Stack
This visualization highlights the hidden costs of manual diagnostics (legal/re-test risks) and how AI specifically targets the high-risk "Error Cost" segment of the budget.

---

## 🏁 Recommendation

**Recommendation: IMPLEMENT**
